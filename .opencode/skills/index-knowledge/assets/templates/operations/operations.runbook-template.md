# Runbooks

This document contains **step-by-step procedures** for handling operational situations.

Runbooks are:

- actionable
- explicit
- optimized for on-call use

---

## Runbook Format

Each runbook should include:

- Trigger
- Symptoms
- Immediate actions
- Verification steps
- Escalation path
